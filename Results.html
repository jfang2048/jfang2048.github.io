Land-Cover Transition: NO₂ / Built Area, PM₂.₅ / Crops, PM₁₀ / Trees

Assigned land-cover binary change masks use code 7 for NO₂ / Built Area, code 5 for PM₂.₅ / Crops, and code 2 for PM₁₀ / Trees. Values 1, 2, and 3 represent Stable class, class Gain, and class Loss; NoData, unrelated transitions, and cloud transitions (code 10) are excluded. Statistics are calculated from the Germany_LCC_2021_2023 EPSG:3035 equal-area raster (QGIS unique values report). Pixel percent and area percent are computed within the Stable + Gain + Loss denominator for each assigned class.

Population Exposure and Bivariate Results

Bivariate population exposure combines pollutant concentration class and population class using bivariate = pol_class_max × 10 + pop_class_median. Each bivariate layer has 402 GAUL Level 2 features, required fields are present, and the formula check returned zero mismatches for all three pollutants. The dissolved chart layers summarize exposure by pol_class_max using pop_sum.
